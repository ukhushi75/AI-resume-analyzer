# AI Resume Analyzer

An AI-powered Resume Analyzer that helps users upload resumes, evaluate them against job listings, and get personalized feedback and ATS scores. Built with React, Next.js, Tailwind CSS, and Puter.js for seamless AI integration and a modern, responsive UI.

---

## Features

- **Resume Upload & Storage**: Users can upload and store multiple resumes securely.  
- **AI Resume Matching**: Evaluate resumes against job descriptions with ATS scoring and customized feedback.  
- **Modern UI/UX**: Clean, responsive design built with Tailwind CSS and reusable components.  
- **Cross-Device Compatibility**: Works seamlessly on desktop, tablet, and mobile devices.  
- **Interactive Chatbot**: Ask questions about your skills, experience, and resume through an AI-powered assistant.  

---

## Tech Stack

- **React / Next.js** – For building the frontend and server-rendered pages.  
- **Puter.js** – Handles serverless auth, storage, and AI integration directly in the browser.  
- **Tailwind CSS** – Utility-first styling for a clean, responsive interface.  
- **TypeScript** – Ensures type safety and better maintainable code.  
- **Zustand** – Lightweight state management.
